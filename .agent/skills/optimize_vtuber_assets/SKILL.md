# optimize_vtuber_assets スキル実行手順

1. **対象画像の特定**
   ユーザーから指定された画像が使用されているコンポーネント（例: `Hero.tsx` や `Profile.tsx` など）を特定し読み込みます。

2. **`next/image` への書き換え**
   通常の `<img>` タグが使われている場合、必ず `next/image` の `<Image>` コンポーネントに書き換えます。

3. **最適化プロパティの完全付与**
   ファーストビューにあたる画像（メインビジュアルの立ち絵等）には、爆速表示のために必ず以下のプロパティを付与して最適化します。
   - `priority`: 即時読み込みを強制し、LCP（Largest Contentful Paint）スコアを極限まで高めます。
   - `width` / `height` または `fill`: レイアウトシフト（CLS）を防ぐため、アスペクト比を固定するか親要素を基準とした絶対配置を行います。
   - `alt`: SEOとアクセシビリティのため、具体的な代替テキスト（例: "〇〇（VTuber名）の全身立ち絵"）を設定します。

4. **画像のフォーマット・サイズについての提案**
   元画像のサイズが数MB以上とあからさまに巨大であると推測される場合は、元画像をWebP形式に変換するかサイズを事前に圧縮することをユーザーに提案（アドバイス）します。

5. **完了報告**
   画像の最適化コーディングが完了したことを伝え、パフォーマンス向上の理由を添えて報告します。
