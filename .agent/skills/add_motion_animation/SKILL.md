# add_motion_animation スキル実行手順

1. **対象コンポーネントの特定**
   アニメーションを付与したいUIコンポーネント群（例: メインビジュアル、セクション見出し、リストなど）を読み込みます。

2. **`motion` パッケージのインポート**
   対象ファイルのトップで `motion` パッケージがインポートされているか確認し、無ければ追加します。
   ```tsx
   import { motion } from "motion/react"; // 最新の記法を使用
   ```

3. **HTMLタグの置換**
   アニメーションさせたい対象のタグ群（`div`, `h1`, `img` 等）を `motion.div`, `motion.h1` 等に書き換えます。

4. **アニメーション変数の定義と適用（Variantsの活用）**
   プロジェクトの「着せ替え」コンセプトや指定されたテーマ（例：ポップ、エレガント）に合わせて、自然なアニメーションプロパティ（`initial`, `animate`, `transition`）を記述します。
   - **例（フェードインアップ）**: `initial={{ opacity: 0, y: 20 }}` -> `animate={{ opacity: 1, y: 0 }}` -> `transition={{ duration: 0.5, ease: "easeOut" }}`
   - 複数要素が連続して出現する場合は `staggerChildren` などを駆使してリッチな「登場感」をプロデュースします。

5. **完了報告**
   付与したアニメーションの動作イメージ（「下からふわっと出てくるようにしました」等）をユーザーに報告し、ブラウザでの表示確認を促します。
